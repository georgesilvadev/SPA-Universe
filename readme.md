## 💻 Projeto

O SPA Universe é um projeto baseado em técnicas de SPA, utilizando HTML, CSS e Javascript puro.

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML
- CSS
- Javascript
- Git e Github
